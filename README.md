# recipe_app

flutter 요리 레시피 앱

클린 아키텍처 기반의 MVVM, MVI 패턴
상태관리 라이브러리 미사용
테스트 가능한 UI 설계
딥 링크 친화적인 라우터 구성
에러 처리를 위한 에러 핸들링
